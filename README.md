# TTC-eLearning-Monitor
TTC eLearning Monitor is a school-based system designed to track and manage student learning progress across courses. The project supports secure login structured data collections (students, courses, progress), and reporting features for monitoring performance at monthly, termly, and annual levels.

## Features
- Student progress tracking
- Library integration
- Smart classroom support
- Easy deployment with GitHub Pages
- custom domain support
